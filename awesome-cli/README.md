# Awesome CLI

A simple command-line interface that prints a random motivational quote to brighten your day.

## Installation

```bash
pip install awesome-cli
```

## Usage

Run the command:

```bash
awesome-cli
```

Example output:

> "Believe you can and you're halfway there." – Theodore Roosevelt

## Development

To install in development mode:

```bash
git clone https://github.com/yourusername/awesome-cli.git
cd awesome-cli
pip install -e .
```

## Running Tests

```bash
pytest
```
