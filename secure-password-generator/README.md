# Secure Password Generator

A simple, secure password generator written in Python that creates strong, random passwords with customizable options.

## Features

- Generate cryptographically secure random passwords
- Customizable length
- Option to include/exclude uppercase letters, lowercase letters, digits, and symbols
- Generate multiple passwords at once
- No dependencies - uses only Python standard library
- Cross-platform (Windows, macOS, Linux)

## Installation

No installation required! Just download the `main.py` file and run it with Python 3.

## Usage

```bash
python main.py [options]
```

### Options

- `-l, --length LENGTH`: Length of the password (default: 16)
- `--no-upper`: Exclude uppercase letters
- `--no-lower`: Exclude lowercase letters
- `--no-digits`: Exclude digits
- `--no-symbols`: Exclude symbols
- `-n, --number NUMBER`: Number of passwords to generate (default: 1)

### Examples

Generate a single 16-character password:
```bash
python main.py
```

Generate a 20-character password:
```bash
python main.py -l 20
```

Generate 5 passwords, each 12 characters long:
```bash
python main.py -l 12 -n 5
```

Generate a password without symbols:
```bash
python main.py --no-symbols
```

Generate a PIN (digits only):
```bash
python main.py --no-upper --no-lower --no-symbols -l 6
```

## Security

This password generator uses Python's `secrets` module, which is designed for generating cryptographically strong random numbers suitable for managing data such as passwords, account authentication, security tokens, and related secrets.

Each password is generated with:
- At least one character from each selected character set
- Cryptographically secure random selection
- Proper shuffling to avoid predictable patterns
