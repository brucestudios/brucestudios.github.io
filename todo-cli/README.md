# Todo CLI

A simple command-line interface for managing your todo list.

## Features

- Add new todos
- List all todos
- Mark todos as complete
- Delete todos
- Persistent storage using JSON

## Installation

```bash
pip install todo-cli
```

## Usage

```bash
# Add a new todo
todo add "Buy groceries"

# List all todos
todo list

# Mark a todo as complete
todo complete 1

# Delete a todo
todo delete 1
```
